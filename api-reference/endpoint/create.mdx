# Pause Subscription API

**Request Endpoint:**

```
POST {url}/router/subscription/pause
```

**Headers:**

```
sign: sha256(RequestBody + "secretkey")
```

**Request Body Example:**

```
{
  "terminalId": "240000840001",
  "subscriptionId": "S24111014002252200004812",
  "pauseFlag": "2",
  "pauseStartDate": "2024-11-12",
  "pauseEndDate": "2024-11-15"
}
```

## Request Parameters

| Field Name     | Type   | Required | Description                                                                        |
| :------------- | :----- | :------- | :--------------------------------------------------------------------------------- |
| terminalId     | String | Y        | Terminal ID provided by Payinsider.                                                |
| subscriptionId | String | Y        | Payinsider subscription ID.                                                        |
| pauseFlag      | String | Y        | Pause type: `1` = Immediate pause, `2` = Custom pause. Required if `pauseFlag=2`.  |
| pauseStartDate | String | C        | Scheduled pause start date (format: YYYY-MM-DD). Required when `pauseFlag` is `2`. |
| pauseEndDate   | String | N        | Scheduled pause end date (format: YYYY-MM-DD).                                     |

## Response Parameters

| Field Name         | Type    | Required | Description                                                      |
| :----------------- | :------ | :------- | :--------------------------------------------------------------- |
| terminalId         | String  | Y        | Terminal ID provided by Payinsider.                              |
| recurringToken     | String  | Y        | Subscription order token value.                                  |
| subscriptionId     | String  | Y        | Payinsider subscription ID.                                      |
| piCustomerId       | String  | Y        | Payinsider subscriber ID.                                        |
| status             | Integer | Y        | Subscription status: `1` = Active, `3` = Canceled, `4` = Paused. |
| cancel             | Boolean | Y        | Whether cancellation is initiated.                               |
| expectedCancelDate | String  | N        | Scheduled cancellation date.                                     |
| pause              | Boolean | Y        | Whether pause was initiated.                                     |
| expectedPauseStart | String  | N        | Scheduled pause start date.                                      |
| expectedPauseEnd   | String  | N        | Scheduled pause end date.                                        |
